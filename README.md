# Wishly — Daily promotions. Zero effort.

Wishly builds daily social media promotions for restaurants. Connected to your Google Business data, it creates ready-to-post images and copy in under 2 minutes.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
